# CSharp_ComparadorWebRequests

## Descrição
O **CSharp_ComparadorWebRequests** é uma aplicação desenvolvida em C# que possibilita a comparação de requisições web. 
Este projeto foi criado com o objetivo de auxiliar desenvolvedores e testers a analisar e verificar diferenças entre diferentes web requests.

## Funcionalidades
- Comparação de parâmetros entre requisições.
- Suporte para requisições HTTP e HTTPS.
- Interface intuitiva para visualização das diferenças.
- Logs detalhados para análise de resultados.

## Tecnologias Utilizadas
- Linguagem: **C#**
